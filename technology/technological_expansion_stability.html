technological stability

Operating things has requirements, such as an engine requiring fuel.

If our entire society and economy has a multitude of things it requires, and it is technologically advancing at the same time, it may be possible to advance to a state where more things and some simpler things are required that are no longer being produced by society, because the production was perceived as "low value".

If these "simple things" are immaterial, it is possible to reach a state of... using physical metaphors, going uphill too fast and losing ground contact on bumps or such.

The most relevant example is probably something like COBOL which we generally don't need, except in the cases where we do. It is "not feasible" to change the system away from using COBOL and it is possible that the demand can't be met either, because education in that area is deprioritized so much it just can't cover the demand anymore.

Specifically with COBOL that won't happen because we're aware of it. I'm concerned with some "unknown known" problems? Groups of problems that can come up but we're not sure in which area?
